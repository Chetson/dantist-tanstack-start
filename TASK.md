## 1. Дизайн-философия: корпоративный портал стоматологической клиники

Сайт должен вызывать ощущение **строгого, надёжного, современного медицинского учреждения**. Никакой анимационной игривости, без «воздушных» градиентов в духе стартапов. Эстетика — **премиальная частная клиника**: швейцарская чистота, немецкая функциональность, скандинавский минимализм.

### Ключевые принципы:
- **Минимализм:** минимум визуального шума, много воздуха (щедрые отступы), чёткая иерархия
- **Строгость:** прямые линии, сдержанные тени, минимум скруглений (радиус `--radius: 0.5rem` вместо текущих 0.625rem)
- **Типографика:** крупный, чистый шрифт; заголовки — насыщенные, текст — с высоким контрастом
- **Структура:** всё разбито на логические блоки с явными разделителями или контрастным фоном
- **Акценты:** один доминантный акцентный цвет; никакой радуги

## 2. Цветовая палитра (на основе `Hero.tsx` + коррективы)

Ориентируемся на цвета, уже использованные в `Hero.tsx` вручную:

| Роль | Hex | OKLCH (примерный) | Применение |
|---|---|---|---|
| **Primary** | `#4A90C4` | `oklch(62% 0.12 240)` | Кнопки, ссылки, акцентные иконки, активные состояния |
| **Primary-foreground** | `#FFFFFF` | `oklch(1 0 0)` | Текст на primary-фоне |
| **Primary-light** | `#6AA8D9` | `oklch(68% 0.10 240)` | Светлые акцентные плашки, фон hover |
| **Primary-pale** | `#C1DEF2` | `oklch(85% 0.04 240)` | Очень светлые фоны секций |
| **Accent / Secondary accent** | `#5F4ACB` | `oklch(48% 0.17 280)` | Важные уведомления (ImportantNotice), левые border-акценты, редкие яркие акценты |
| **Background** | `#FAFBFD` | `oklch(98.5% 0.003 250)` | Фон страницы: не чисто-белый, а с лёгким холодным оттенком |
| **Foreground** | `#0F172A` | `oklch(18% 0.03 260)` | Основной текст: глубокий тёмно-синий, не чёрный |
| **Card** | `#FFFFFF` | `oklch(1 0 0)` | Белые карточки |
| **Muted** | `#F1F5F9` | `oklch(96.5% 0.005 250)` | Серый фон для выделения блоков |
| **Muted-foreground** | `#64748B` | `oklch(55% 0.03 255)` | Второстепенный текст |
| **Border** | `#E2E8F0` | `oklch(90% 0.01 250)` | Границы и разделители |
| **Destructive** | `#DC2626` | `oklch(55% 0.22 27)` | Ошибки и предупреждения об опасности |

### Изменения в `app.css`

Заменить CSS-переменные в `:root` на новую палитру. ВАЖНО: Tailwind v4 требует, чтобы переменные были в OKLCH (или OKLCH-совместимых) форматах для корректной работы opacity-модификаторов типа `bg-primary/15`.

**Текущий primary:** `oklch(61.8% 0.138 244.8)` → заменить на `oklch(62% 0.12 240)`
**Текущий background:** `oklch(1 0 0)` → заменить на `oklch(98.5% 0.003 250)`
**Текущий foreground:** `oklch(0.129 0.042 264.695)` → заменить на `oklch(18% 0.03 260)`
**Текущий muted:** `oklch(0.968 0.007 247.896)` → заменить на `oklch(96.5% 0.005 250)`
**Текущий muted-foreground:** `oklch(0.554 0.046 257.417)` → заменить на `oklch(55% 0.03 255)`
**Текущий border:** `oklch(0.929 0.013 255.508)` → заменить на `oklch(90% 0.01 250)`

**Радиус скругления:**
```
--radius: 0.5rem;  /* было 0.625rem */
```

## 6. Общие правила для всех компонентов

### Карточки (Card)
- Везде заменить `transition-shadow hover:shadow-md` на `border border-border shadow-none` (или лёгкую `shadow-xs`)
- Внутренние отступы: `CardHeader` — `px-6 pt-6 pb-2`, `CardContent` — `px-6 pb-6`

### Разделители
- Использовать `Separator` для явного разделения логических блоков
- Цвет: `bg-border`

### Кнопки
- `Button variant="default"`: `bg-primary text-primary-foreground hover:bg-primary/90`
- `Button variant="outline"`: `border-border text-foreground hover:bg-muted`
- `Button variant="ghost"`: как сейчас — для навигации
- Добавить `radius-sm` (определено в теме через `--radius-sm`)

### Максимальная ширина контента
- `container mx-auto` с `max-w-7xl` (или определяется автоматически через Tailwind container)
- Горизонтальные отступы: `px-6 sm:px-12` (оставить как есть)

### Сетка секций
- Отступы между секциями: `py-20 sm:py-24` (щедрые, но не чрезмерные)

## 9. Резюме: что должно получиться

Строгий корпоративный портал стоматологической клиники: бело-серо-синяя гамма, крупная типографика, щедрые отступы, никаких лишних украшений. Единственный яркий элемент — фиолетовый баннер предупреждения `ImportantNotice`. Сайт выглядит как внутренний портал премиальной клиники, куда пациент заходит за проверенной информацией, а не как лендинг с маркетинговыми уловками.

**Цветовой аккорд:** `primary` (синий `#4A90C4`) + `accent` (фиолетовый `#5F4ACB`) + серый `muted` (`#F1F5F9`) + тёмно-синий текст (`#0F172A`).
